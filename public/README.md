# ☀️ SunCart – Summer Essentials Store

## Project Name
**SunCart** – An e‑commerce platform for summer products like sunglasses, outfits, skincare, and beach accessories.

## Purpose
SunCart allows users to browse summer essentials, view product details (only after login), and manage their profile. It’s designed to demonstrate authentication, protected routes, responsive design, and a modern UI with smooth animations.


## Key Features
- 🔐 **User Authentication** – Email/password (demo) + Google OAuth.
- 🛡️ **Protected Routes** – Product details and profile pages require login.
- 🏠 **Home Page** – Hero slider, popular products, summer skincare tips, top brands.
- 📦 **Products Page** – Grid layout with all summer items.
- 👤 **My Profile** – View profile info (name, email, photo) and update name/photo.
- 📱 **Fully Responsive** – Works perfectly on mobile, tablet, and desktop.
- ✨ **Animations** – Animate.css used for entrance animations (bounceIn, fadeInUp, etc.).
- 🔄 **Environment Variables** – `.env.local` used for NextAuth secrets and Google credentials.

## NPM Packages Used
- `next` – React framework
- `next-auth` – Authentication (Credentials + Google)
- `react` & `react-dom` – UI library
- `tailwindcss` & `daisyui` – Styling and pre‑built components
- `react-hot-toast` – Toast notifications
- `animate.css` – Pre‑built CSS animations
- `@heroicons/react` – Icons (hamburger menu, etc.)
